# CBZ-scripts
Scripts for creating, bundling, and converting CBZ files.

主にCBZを扱うためのスクリプト群です。

CBZ形式は厳密に言えばメタデータをXMLで格納したり色々な定義があるかと思われますが、
ここでは「ZIP形式アーカイブの拡張子を書き換えただけのモノ」と捉えて取り扱っています。  
そのため、特に複雑なことはしていない、シンプルなスクリプト群であるハズです。

その割になぜかやたらと行数が多いのが混じってるのはなぜだろう。  
おおかたAIに書いてもらったシロモノなのでよくわかりません。


## 本パッケージの説明

それぞれバッチスクリプトやPowershellスクリプトで作成されています。

基本的にはスクリプトを処理対象のディレクトリに置いて実行するだけです。

一部のスクリプトは組となるスクリプトや、
ツールの入ったディレクトリごとまとめて配置する必要があります。  
基本的には自分用に作った間に合わせのものなので、
使いやすさとかは諦めてください。


## 各ツールの説明

### CBZ-converter
開封が重たいZIP以外のアーカイブ形式を展開してCBZに変換します。


### CBZ-merge
分割ダウンロードにありがちな連番アーカイブのコミックブックを結合します。


### CBZ-unpacker
CBZを一斉に解凍するためのスクリプトです。  


### CBZ-upscaler
高解像度閲覧用の軽量データを作成するスクリプトです。  

ディレクトリにあるCBZを全て展開した後、
展開先を形式やサイズに応じて適宜拡大処理をかけて、
Webp形式に変換したのちCBZに戻します。


### EXT-renamer
拡張子を書き換えるだけのシンプルなスクリプト群です。


### PDFtoIMG
スクリプトにPDFをドラッグアンドドロップすることで、
Popplerを使ってPDFを画像に分解することができます。

pdf2cbrはPDFから展開した画像をそのままCBZに再梱包しますが、
大抵は余計な白コマや黒コマが出力されているなど、
閲覧にあたって調整が必要な状態になっていることが殆どなので、
ほとんど実用性はありません。

基本的にはpdf2dirを利用することになると思います。
